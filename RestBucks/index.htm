This is an open source self-documented example application of a level-3 REST api based on the excelent example of Rest in Practice. This example is a port to Nancy of the Restbucks on .NET example Get some rest.

Ordering coffee here!

You can use this site to try out the application easily. Follow these steps:

  • 1 st Read how to create an order.
  • 2 nd Execute a POST with fiddler and grab the location of your order.
  • 3 rd GET the order and follow the links and the documentation for each operation.

Further reading...

  • Read the article How to GET a Cup of Coffee at infoq.
  • Buy the book.
